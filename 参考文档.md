# salesPredict

### ARIMA模型简介
ARIMA模型提供了基于时间序列理论，对数据进行平稳化处理（AR和MA过程）、模型定阶（自动差分过程）、参数估计，建立模型，并对模型进行检验。
在Python中statsmodels就囊括了时间序列过程。

### 环境
- Windows 10
- Python 3.6.5

### 依赖包
```
pip install -r requirements.txt
```

### 程序执行
```
python sales.py
```
